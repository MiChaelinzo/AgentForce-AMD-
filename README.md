## 🧠 Overview  
AgentForce‑AMD is a next‑generation AI agent framework built on **Bolt.new**, designed to streamline modular workflows, accelerate innovation, and showcase **AMD‑powered applications** in real‑world scenarios.  
This project was developed for the **AMD Developer Hackathon**, demonstrating how adaptive agents and high‑performance compute can converge to enable scalable, intelligent automation.

---

## 🚀 Features  
- **Modular Agent Architecture** — Plug‑and‑play components for reasoning, data ingestion, and action execution.  
- **Real‑Time Monitoring** — GPU metrics (utilization, VRAM, temperature, power) displayed via AMD APIs.  
- **Multi‑Agent Workflows** — Orchestrate Llama 3.2, DeepSeek Coder, Qwen 2.5/VL, and Mistral 7B models.  
- **Edge Functions for AI Inference** — Execute tasks and simulate agent responses dynamically.  
- **Workflow Builder UI** — Drag‑and‑drop configuration for orchestration pipelines.  
- **Secure Authentication** — Bolt Database Auth integration for user management.  
- **AMD‑Inspired Design** — Dark theme with red accents (#e8001d), glassmorphism, and responsive layout.

## 🧪 Setup & Deployment  
1. Clone the repository:  
   ```bash
   git clone https://github.com/MiChaelinzo/AgentForce-AMD-.git
   cd AgentForce-AMD-
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Run locally:  
   ```bash
   npm run dev
   ```
4. Deploy via **Bolt.new** or **StackBlitz** for instant hosting.

## 🧰 Core Modules  
- **Agents** — Manage AI models, view capabilities, and performance stats.  
- **Workflows** — Create multi‑agent orchestration pipelines with triggers (manual, webhook, scheduled).  
- **Dashboard** — Monitor GPU performance and agent activity in real time.  
- **Tasks** — Track execution feeds and performance metrics.
